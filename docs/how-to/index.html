How-To Guides

Step-by-step walkthroughs for common tasks. Pick a guide and follow along.

Hardware

Prepare a Raspberry Pi

Use Raspberry Pi Imager to flash an SD card, set a hostname, configure Wi-Fi or Ethernet, and get your Pi online and ready for FD Commander.

Getting Started

SSH & Command-Line Basics

Connect to your Raspberry Pi over SSH, learn essential Linux commands, check on FD Commander, copy files with scp, and keep sessions alive with tmux.

Networking

Set Up a LAN for Field Operations

Build a reliable local network for your Field Day site using a router, switch, or a Raspberry Pi. Covers wired and wireless setups, IP addressing, and DNS.

Advanced · Optional

Email Setup

Enable password resets, invitation emails, shift reminders, and notifications by adding SMTP credentials to your .env file.

Advanced · Optional

DNS & TLS Setup

Point a domain name at your server and add HTTPS with Let's Encrypt. Only needed if you want to put FD Commander on the internet for remote access.
